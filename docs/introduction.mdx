## Bem-vindo à API Tanque Cheio

A API Tanque Cheio é uma solução robusta e escalável, construída com **NestJS 11** e **TypeORM**, para fornecer dados atualizados sobre preços de combustíveis e informações de postos em todo o Brasil.

Nossa missão é oferecer uma plataforma confiável para desenvolvedores criarem aplicações inovadoras, desde aplicativos para consumidores até análises de mercado complexas.

### Principais Funcionalidades
- **Consulta de Postos**: Busque postos de combustível por geolocalização, cidade ou CNPJ.
- **Preços em Tempo Real**: Acesse o histórico e os preços atuais dos combustíveis.
- **Sincronização de Dados**: Um sistema automatizado para sincronizar dados da Agência Nacional do Petróleo (ANP).
- **Autenticação Segura**: Baseada em JWT para garantir que apenas usuários autorizados acessem os dados.

Esta documentação guiará você através da configuração inicial, autenticação, e como utilizar todos os nossos endpoints.
